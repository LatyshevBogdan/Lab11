# Lab11
0. Установка всех утилит.
1. Поднятие ngrok.
```
./ngrok tcp 22
```
2. Проверка сетевого соединеня.
```
nc -v <домен> <порнт>
```
3. Запуск ssh для удаленного доступа.
```
sudo service ssh start
```
4. Подключаемся к нашей же машине через сеть
```
ssh <имя пользователя><домен> -p<порт>
```
